# Python Service Installer - Альтернативное решение

## Проблема с PowerShell
PowerShell скрипт `install-all-services.ps1` имел множественные синтаксические ошибки:
- Неправильные фигурные скобки
- Отсутствующие блоки catch
- Структурные проблемы с функциями

## Python решение
Создан альтернативный скрипт на Python `install-services.py` с преимуществами:
- ✅ Четкий и понятный синтаксис
- ✅ Лучшая обработка ошибок
- ✅ Кроссплатформенность
- ✅ Более надежная работа

## Использование

### Прямой запуск Python:
```bash
python install-services.py status
python install-services.py install
python install-services.py start
python install-services.py stop
python install-services.py restart
```

### Через batch файл:
```bash
install-services.bat status
install-services.bat install
install-services.bat start
install-services.bat stop
install-services.bat restart
```

## Команды

- `status` - Показать статус всех служб
- `install` - Установить все службы (требует права администратора)
- `uninstall` - Удалить все службы (требует права администратора)
- `start` - Запустить все службы
- `stop` - Остановить все службы
- `restart` - Перезапустить все службы

## Требования

- Python 3.6 или выше
- Права администратора для установки/удаления служб

## Преимущества Python версии

1. **Надежность**: Нет проблем с синтаксисом PowerShell
2. **Читаемость**: Код легче понимать и поддерживать
3. **Обработка ошибок**: Лучшая обработка исключений
4. **Кроссплатформенность**: Работает на разных ОС
5. **Простота**: Меньше сложности в структуре кода

## Статус служб

Скрипт показывает статус каждой службы:
- **Running** - Служба запущена
- **Stopped** - Служба остановлена
- **Starting** - Служба запускается
- **Stopping** - Служба останавливается
- **Not Installed** - Служба не установлена
