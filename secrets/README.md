# Secrets Directory

**Обновлено**: 5 ноября 2025, 20:15

Эта директория содержит секретные данные для CI/CD системы.

## Файлы

| Файл | Описание | Значение | Статус |
|------|----------|----------|--------|
| `gitsync_password.txt` | Пароль пользователя хранилища 1С | `123` | ✅ Активен |
| `gitlab_token.txt` | Personal Access Token GitLab | `YOUR_GITLAB_TOKEN_HERE` | ✅ Активен |
| `redmine_password.txt` | Пароль администратора Redmine | `admin` | ⚠️ По умолчанию |

## Использование

Секреты монтируются в контейнеры как файлы и читаются при запуске сервисов.

### GitSync Password
```yaml
GITSYNC_STORAGE_PASSWORD: "123"
```

### GitLab Token
```yaml
GITLAB_TOKEN: "YOUR_GITLAB_TOKEN_HERE"
```

### Redmine Password
```yaml
REDMINE_PASSWORD: "admin"
```

## Безопасность

⚠️ **ВАЖНО**:
- НЕ коммитьте эту директорию в Git
- Файлы секретов добавлены в `.gitignore`
- Меняйте пароли по умолчанию в production
- Используйте сложные пароли

## Ротация Секретов

Для обновления секретов:
1. Измените содержимое файла
2. Перезапустите контейнеры: `docker-compose restart`
3. Обновите документацию
